# Task 6 - Password Strength Evaluation

## Objective

To create passwords with varying levels of complexity, evaluate their strength using an online password strength checker, and understand password security best practices.

## Tools Used

* PasswordMeter

## Files Included

* Password_Strength_Report.md
* Screenshots folder containing password strength results

## Steps Performed

1. Created multiple passwords with different complexity levels.
2. Tested each password using an online password strength checker.
3. Recorded the strength results.
4. Captured screenshots of the evaluations.
5. Researched common password attacks.
6. Documented password security best practices.

## Outcome

This task improved understanding of password security, common attack methods, and techniques for creating strong passwords.
