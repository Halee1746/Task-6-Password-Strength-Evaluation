# Password Strength Evaluation Report

## Objective

To understand the characteristics of strong passwords and evaluate password strength using an online password strength checker.

## Tools Used

* PasswordMeter (Online Password Strength Checker)

## Observations

* Passwords containing common words were rated very weak.
* Adding uppercase letters improved strength.
* Including special characters and numbers significantly increased security.
* Longer passwords received higher strength ratings.
* Randomly generated passwords were the strongest.

## Common Password Attacks

### Brute Force Attack

A brute force attack attempts every possible password combination until the correct one is found.

### Dictionary Attack

A dictionary attack uses a list of commonly used passwords and words to guess passwords.

### Credential Stuffing

Attackers use usernames and passwords leaked from previous data breaches to gain unauthorized access.

### Phishing

Attackers trick users into revealing passwords through fake emails or websites.

## Best Practices for Creating Strong Passwords

* Use at least 12–16 characters.
* Combine uppercase and lowercase letters.
* Include numbers and special symbols.
* Avoid common words and predictable patterns.
* Do not use personal information.
* Use a unique password for each account.
* Enable two-factor authentication (2FA).
* Consider using a password manager.

## Conclusion

Password complexity has a significant impact on security. Longer passwords containing uppercase letters, lowercase letters, numbers, and special characters are much more resistant to brute force and dictionary attacks. Following password security best practices helps protect accounts from unauthorized access.
